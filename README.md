# E-Commerce-Tecnologico :calling:
## La idea de este proyecto es desarrollar el Front de un Ecommerce de venta Tecnologica.

###   Productos como:
* Cargadores de teléfonos.
* Cables para celulares.
* Airpods, Audifonos Bluetooth. :speaker:
* Iphones :iphone:
* Entre otros. (Productos similares)

###   Paleta de colores:
* Ojalá colores pasteles / Pop.
* Por definir/discutir.

###   Sistema Responsivo:
* Ideal que el código se desarrolle simultaneamente la versión Web 1980px y versión celular. 
* Testeo de código.

###   Otros:
* Modelo simple, fácil de digerir, información precisa.
